## 내가 맡은 페이지 
- 메인 페이지
  1. 배너 이미지 자동 슬라이드
  2. 애견 카페 검색 이동
  3. 인기 애견카페 추천
  4. 인기글 추천 
- 카페맵 페이지
  1. 카페 지도 검색
  2. 카카오 API
- 카페 상세 페이지
  1. 카페 리뷰 모아보기
  2. 평균 별점
- 헤더 / 푸터
  1. 네비게이터
